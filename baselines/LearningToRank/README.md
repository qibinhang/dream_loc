### How to train
1. run `python prepare_data.py tomcat`
2. run `python api_info_extractor.py tomcat`
3. run `python api_enrich_sim.py tomcat`
4. run `python class_name_sim.py tomcat`
5. run `python dataset.py tomcat`
6. run `python svm.py tomcat`


### How to use pre-trained model
1. run `python svm.py tomcat just_test`
